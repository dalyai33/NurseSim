=====================================
         NurseSim+ Setup Guide
=====================================
Last updated: November 2025
Team: NurseSim+
-------------------------------------

[ STEP 1: INSTALL REN'PY ]
-------------------------------------
1. Go to: https://www.renpy.org/latest.html
2. Download and install the latest version of Ren'Py
   for your operating system.

[ STEP 2: CLONE THE PROJECT ]
-------------------------------------
1. Open your terminal or Git Bash.
2. Run this command:
      git clone https://github.com/dalyai33/NurseSim.git
3. Open the Ren'Py application (renpy.exe).

[ STEP 3: RUN THE SIMULATOR ]
-------------------------------------
1. In the Ren'Py launcher, click on Preferences->Project Directory
2. Set the Project Directory to the Git Repository Folder
3. Select on "renpyNurseSim" and Click "Launch Project" to start the simulator.

[ STEP 4: EDIT THE PROJECT ]
-------------------------------------
- Open these files to make changes:
    script.rpy   -> main story and dialogue
    options.rpy  -> settings and configurations
    screens.rpy  -> menus and UI elements
- You can edit with any text editor or Ren'Pyâ€™s built-in editor.
- If using Visual Studio, make sure to download Ren'Py Language Reader extension
